K Upload :: SilentDownload

XML : <upload_transfer_window><silent_download>.....</silent_download></upload_transfer_window>
Script : SilentDownload

Script 설정이 우선 적용됩니다.
다운로드 전송창 노출 여부를 설정합니다.

remarks

기본값은 "0" 노출이고, "1"로 설정시 미노출, "2"로 설정시 일시정지,취소버튼 미노출(Agent)입니다.

sample code

다운로드 전송창을 미노출합니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        .....
        <silent_download>1</silent_download>
        .....
    </upload_transfer_window>
    .....
</setting>
.....
