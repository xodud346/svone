K Upload :: UseBeforeOpenEvent

XML : <use_before_open_event>.....</use_before_open_event>, Script : UseBeforeOpenEvent

Script 설정이 우선 적용됩니다.
파일 열기 전 이벤트 함수 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 이벤트 미사용합니다.
"1"으로 설정 후 파일 열기 전
RAONKUPLOAD_BeforeOpenFile(uploadID, paramObj) 함수가 호출 됩니다.

이 함수 안에서 return true 이면 파일 열기, return false 이면 파일 열기를 하지 않습니다.

sample code

파일 열기 전 이벤트 함수 미사용으로 설정합니다.

XML

.....
<setting>
    .....
    <use_before_open_event>0</use_before_open_event>
    .....
</setting>

Event Script

<script type="text/javascript">
    function RAONKUPLOAD_BeforeOpenFile(uploadID, paramObj){
        var returnValue = false;
        if(파일 열기 조건){
            returnValue = true;
        }
        return returnValue;
    }
</script>
