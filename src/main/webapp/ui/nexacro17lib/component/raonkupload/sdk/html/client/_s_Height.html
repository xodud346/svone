K Upload :: Height

XML : <height>.....</height>, Script : Height

Script 설정이 우선 적용됩니다.
업로드의 높이를 설정합니다.

remarks

기본값은 "200px" 입니다.
최소 높이값은 "120px" 입니다.
높이는 "px" 만 가능합니다.

sample code

업로드의 높이값을 400px 으로 설정합니다.

XML

.....
<setting>
    .....
    <height>400px</height>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            Height: '400px'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
