K Upload :: MultiTransfer

void MultiTransfer()

멀티업로드를 사용하는 경우 멀티업로드 전송 시작을 동작합니다.

return value

없음.

parameters

없음.

remarks

멀티업로드를 사용하는 경우 하나의 버튼으로 전체를 전송하고 싶은 경우 유용합니다.

            RAONKUPLOAD.MultiTransfer();
