K Upload :: SetCustomHeaderByIndex

object SetCustomHeaderByIndex(rowIndex, customHeaderValue, uploadID)

사용자 헤더를 추가한 경우 파일의 순서 정보로 사용자 헤더에 추가한 항목의 값을 추가합니다.

return value

object  순서 정보에 해당하는 객체를 리턴합니다.

parameters

rowIndex  0부터 시작되는 파일의 순서 정보를 의미합니다.
customHeaderValue  사용자 헤더에 추가한 항목의 값을 의미 합니다
uploadID  실행할 업로드의 id를 의미합니다.

remarks

추가되어있는 파일 리스트에 사용자 헤더에 추가한 항목의 값을 추가하려는 경우 유용합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">          
        function setCustomHeaderByIndex() {
            RAONKUPLOAD.SetCustomHeaderByIndex("0", "CustomHeaderValue", "upload1");
        }
</script>	

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new RAONKUpload({Id: 'upload1'});
        </script>
</div>
