K Upload :: SetViewerBase64JsPath

void SetViewerBase64JsPath(String pViewerBase64JsPath)

웹표준 모드의 뷰어사용시 뷰어에 사용되는 kupload/js/raonkupload.base64.js 파일의 경로를 설정합니다.
일반적으로 kupload/handler/raonkviewer.jsp를 다른 경로에 사용될 경우 설정합니다.
이 경우 raonkviewer.jsp 페이지에서 raonkupload.base64.js의 경로가 접근 가능하도록 설정해야 합니다.

return value

없음.

parameters

pViewerBase64JsPath  웹표준 모드의 뷰어사용시 뷰어에 사용되는 kupload/js/raonkupload.base64.js 파일의 경로를 의미합니다.

remarks

기본값 : ../js/raonkupload.base64.js ( kupload/js/raonkupload.base64.js )

sample code

//raonkviewer.aspx에서 설정

RAONKHandler upload = new RAONKHandler();

upload.settingVo.SetViewerBase64JsPath("../js/raonkupload.base64.js");

upload.Process(context);
