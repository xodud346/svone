방법 설명

1. K Upload에 파일 추가할 때 각 파일에 특정 정보를 추가 후 전송하는 방법

K Upload 파일 추가 후 이벤트 함수인 RAONKUPLOAD_AfterAddFile에서
해당 파일에 Custom 정보를 추가할 수 있는 API인 SetFileCustomValue로 설정

<script type="text/javascript">
function RAONKUPLOAD_AfterAddFile(uploadID, paramObj) {
    RAONKUPLOAD.SetFileCustomValue("-1", "CustomValue", uploadID);
}
</script>

RAONKUPLOAD_AfterAddFile 이벤트 함수의 파라미터 정보
uploadID : 이벤트가 발생된 업로드의 ID
paramObj.strName : 추가되는 파일의 파일명
paramObj.nSize : 추가되는 파일의 사이즈
paramObj.nIndex : 0부터 시작되는 파일의 순서 정보
paramObj.strPath : 추가되는 파일의 로컬 경로 정보

RAONKUPLOAD.SetFileCustomValue API의 파라미터 정보
1번째 파라미터 : 0부터 시작되는 파일의 순서정보, -1입력시 마지막에 추가된 파일
2번째 파라미터 : 해당 파일에 설정할 CustomValue 정보
3번째 파라미터 : CustomValue 정보를 추가할 업로드의 ID

2. 서버에서 그 특정 정보를 받는 방법

.NET

string strCustomValue = eventVo.GetCustomValue();

JAVA

string strCustomValue = eventVo.getCustomValue();
