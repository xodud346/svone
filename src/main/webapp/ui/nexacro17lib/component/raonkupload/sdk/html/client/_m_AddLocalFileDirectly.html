K Upload :: AddLocalFileDirectly

void AddLocalFileDirectly(strFileFullPathName, callbackFunction, uploadID)

로컬파일을 업로드 영역에 추가합니다.
로컬경로를 배열로 추가 시엔 AddLocalFileDirectlyEx를 사용합니다.

return value

없음.

parameters

strFileFullPathName  추가할 파일의 로컬 경로를 의미합니다.
callbackFunction  파일 추가 완료 후 발생되는 콜백 함수를 의미합니다.
uploadID  실행할 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function addLocalFileDirectly() {

            var strFileFullPathName = "파일경로";
            
            .....
            
            var callbackFunction = function() {
                //모든 파일 추가 완료 후 전송
                RAONKUPLOAD.Transfer('upload1');
            }
            
            RAONKUPLOAD.AddLocalFileDirectly(strFileFullPathName, callbackFunction, 'upload1');
            
        }
</script>
