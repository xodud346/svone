K Upload :: UseSizeColumn

XML : <use_size_column>.....</use_size_column>, Script : SizeColumnWidth

Script 설정이 우선 적용됩니다.
업로드 영역 항목 중 파일크기 항목 노출 여부를 설정합니다.

remarks

XML:use_size_column  설정값 "0" 파일크기 항목 노출안함이고, "1"로 설정시 파일크기 항목을 노출합니다.
Script:SizeColumnWidth  설정값 "0" 파일크기 항목 노출안함이고, 다른 값으로 설정시 파일크기 항목을 노출 후 해당 넓이를 설정합니다.

sample code

파일크기 항목 노출안함으로 설정합니다.

XML

.....
<setting>
    .....
    <use_size_column>0</use_size_column>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SizeColumnWidth: '0'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
