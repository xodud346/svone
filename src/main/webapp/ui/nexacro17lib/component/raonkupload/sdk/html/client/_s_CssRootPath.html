K Upload :: CssRootPath

XML : <skin_name css_root_path="....."></skin_name>, Script : CssRootPath

Script 설정이 우선 적용됩니다.
사용자가 만든 css 폴더 경로를 업로드 영역에 설정합니다.

remarks

raonkupload > css 폴더 안에 존재 하는 파일명은 유지 시켜주셔야 합니다. (파일명 변경 불가)
http://를 포함하는 전체 url을 입력해야 합니다.

sample code

사용자가 만든 css_admin 폴더 경로를 설정합니다.

XML

.....
<setting>
    .....
    <skin_name css_root_path="http://localhost/css_admin/">.....</skin_name>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            CssRootPath: 'http://localhost/css_admin/'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
