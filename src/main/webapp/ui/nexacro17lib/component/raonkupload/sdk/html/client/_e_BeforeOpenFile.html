K Upload :: BeforeOpenFile

boolean RAONKUPLOAD_BeforeOpenFile(uploadID, paramObj)

파일 열기 전 발생합니다.

return value

boolean  true일 경우 파일 열기를 진행하고, false일 경우 열기하지 않습니다.

parameters

uploadID  파일 열기 요청한 업로드의 id를 의미합니다.
paramObj  열기 실행될 파일에 대한 정보가 있는 object 입니다.
- paramObj.strIsWebFile  Web파일과 Local 파일의 구분 값을 의미합니다. (0:local, 1:web)
- paramObj.strKey  파일의 Unique Key를 의미합니다.
- paramObj.strName  원본 파일명을 의미합니다.
- paramObj.strPath  파일의 경로를 의미합니다.
- paramObj.strIsLargeFile  파일 대용량을 의미합니다. (0:일반, 1:대용량)

remarks

K Upload 설정 config.xml에서 use_view_or_open_event 값이 1일 경우 발생합니다.
