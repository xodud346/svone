K Upload :: DownloadCancel

void RAONKUPLOAD_DownloadCancel(uploadID, paramObj)

파일 다운로드 취소했을 때 발생합니다.

return value

없음.

parameters

uploadID  다운로드 취소된 업로드의 id를 의미합니다.
paramObj  다운로드 취소 관련 정보가 있는 object 입니다.
- paramObj.arrDownloadedFileList  다운로드 취소 시 파일들의 Array 객체를 의미합니다.
- paramObj.arrDownloadedFileList[index].originName  파일의 원본 파일명을 의미합니다.
- paramObj.arrDownloadedFileList[index].fileSize  파일의 사이즈를 의미합니다.
- paramObj.arrDownloadedFileList[index].status  파일의 상태를 의미합니다.
- paramObj.arrDownloadedFileList[index].order  파일의 순서 정보를 의미합니다.
- paramObj.arrDownloadedFileList[index].uniqueKey  파일의 uniqueKey를 의미합니다.

remarks

없음.
