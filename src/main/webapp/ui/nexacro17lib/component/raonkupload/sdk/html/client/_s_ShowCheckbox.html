K Upload :: ShowCheckBox

XML : <show_checkbox>.....</show_checkbox>, Script : ShowCheckBox

Script 설정이 우선 적용됩니다.
업로드 영역의 Check Box 노출 관련하여 설정합니다.

remarks

기본값이 "1" 리스트+Header 사용이고, "0"은 리스트에만 사용입니다.
파일 선택 동작 관계로 리스트의 Check Box는 반드시 노출되어야 합니다.

sample code

업로드 영역의 Check Box를 리스트에만 노출합니다.

XML

.....
<setting>
    .....
    <show_checkbox>0</show_checkbox>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ShowCheckBox: '0'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
