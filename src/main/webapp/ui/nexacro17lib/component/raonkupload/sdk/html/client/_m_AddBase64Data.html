K Upload :: AddBase64Data

void AddBase64Data(strBase64Data, fileName, uploadID)

Base64 이미지 데이터를 전송(업로드)하기 위하여 전송대상 목록에 추가합니다.

return value

없음.

parameters

strBase64Data  Base64 이미지 문자열을 의미합니다.
fileName  이미지 파일명을 의미합니다.
uploadID  실행할 업로드의 id를 의미합니다.

remarks

없음.
