K Upload :: FileNameRuleEx

XML : <file_name_rule_ex>.....</file_name_rule_ex>, Script : FileNameRuleEx

Script 설정이 우선 적용됩니다.
업로드 되는 파일명이 중복될 때 파일명 뒤에 기호를 추가하여 생성됩니다.

remarks

기본값은 "_" 입니다.
기호는 "_" 또는 "#" 으로 설정하면 해당 문자가 파일명 뒤에 붙어 저장됩니다.
값을 "i" 로 설정시 파일명 뒤에 "숫자"가 자동 증가합니다.

sample code

파일명이 중복될 때 숫자로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <file_name_rule_ex>i</file_name_rule_ex>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            FileNameRuleEx: 'i'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
