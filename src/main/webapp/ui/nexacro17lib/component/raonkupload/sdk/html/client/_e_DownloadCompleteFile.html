K Upload :: DownloadCompleteFile

void RAONKUPLOAD_DownloadCompleteFile(uploadID, paramObj)

파일 다운로드 완료 후 발생합니다.

return value

없음.

parameters

uploadID  다운로드가 발생한 업로드의 id를 의미합니다.
paramObj  다운로드 완료된 파일에 대한 정보가 있는 object 입니다.
- paramObj.strPath  파일 경로를 의미합니다.
- paramObj.strDownloadedPath  로컬에 저장된 경로를 의미합니다.
- paramObj.strType  다운로드일 경우 "download"를 의미합니다.
- paramObj.customValue  다운로드된 파일의 customValue를 의미합니다.
- paramObj.size  다운로드된 파일의 사이즈를 의미합니다.
- paramObj.originalName  다운로드된 파일의 원본 파일명을 의미합니다.

remarks

K Upload 설정 config.xml에서 use_download_complete_event의 값이 1일 경우 발생합니다.
