K Upload :: setViewerJsPath

void setViewerJsPath(String pViewerJsPath)

웹 표준 모드의 뷰어 사용시 뷰어에 사용되는 kupload/js/raonkupload.viewer.js 파일의 경로를 설정합니다.
일반적으로 kupload/handler/raonkviewer.jsp를 다른 경로에 사용될 경우 설정합니다.
이 경우 raonkviewer.jsp 페이지에서 raonkupload.viewer.js의 경로가 접근 가능하도록 설정해야 합니다.

return value

없음.

parameters

pViewerJsPath  웹 표준 모드의 뷰어 사용시 뷰어에 사용되는 kupload/js/raonkupload.viewer.js 파일의 경로를 의미합니다.

remarks

기본값 : ../js/raonkupload.viewer.js ( kupload/js/raonkupload.viewer.js )

sample code

//raonkviewer.jsp에서 설정

RAONKHandler upload = new RAONKHandler();

upload.settingVo.setViewerJsPath("../js/raonkupload.viewer.js");

String result = upload.Process(request, response, application, event);
