K Upload :: SetTempPath

void SetTempPath(string pTempPath)

업로드 또는 다운로드시 임시 파일에 대한 폴더를 설정하기 위하여 사용합니다.

return value

없음.

parameters

pTempPath  임시 파일에 대한 폴더경로를 의미합니다.

remarks

기본값 : kupload/handler/temp
SetVirtualPath 또는 SetPhysicalPath의 경로가 공유 드라이브 일 경우
SetTempPath도 해당 공유 드라이브의 위치로 설정되어야 합니다.

sample code

RAONKHandler upload = new RAONKHandler();

upload.settingVo.SetTempPath(@"D:\temp");

upload.Process(context);
