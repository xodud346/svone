K Upload :: GetCustomErrorMessage

string GetCustomErrorMessage(string pCustomErrorCode, string pCustomErrorMessage)

Process를 실행하기 전에 특정 상황에 대하여 에러 처리를 할 경우 사용합니다.
Session Check 등등의 경우에 사용이 가능합니다.

return value

설정된 Error 코드와 Message를 kupload에서 사용되는 암호화된 문자열로 처리 후 return

parameters

pCustomErrorCode  설정하려는 Error의 코드(900이상의 3자리로 설정)를 의미합니다.
pCustomErrorMessage  설정하려는 Error의 Message를 의미합니다.

remarks

없음.

sample code

RAONKHandler upload = new RAONKHandler();

if(인증이 맞지 않을 경우) {
    string strErrorValue = upload.getCustomErrorMessage("999", "사용자오류");
    context.Response.Clear();
    context.Response.Write(strErrorValue);
} else {
    upload.Process(context);
}
