K Upload :: MultiFileSelect

XML : <multi_file_select>.....</multi_file_select>, Script : MultiFileSelect

Script 설정이 우선 적용됩니다.
업로드에서 파일 추가시 여러개 파일 선택 또는 한개씩만 선택 후 적용하도록 설정합니다.

remarks

기본값은 "1" 이고 여러 개의 파일 선택할 수 있도록 설정합니다. "0" 으로 설정시 한개씩만 설정합니다.

sample code

파일선택 시 단일 파일 선택모드로 설정합니다.

XML

.....
<setting>
    .....
    <multi_file_select>0</multi_file_select>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            MultiFileSelect: '0'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
