K Upload :: SetFileCustomValue

void SetFileCustomValue(itemIndex, value, uploadID)

파일을 추가할 때, 각 파일에 특정 정보를 추가 후 전송합니다.

return value

없음.

parameters

itemIndex  0부터 시작되는 파일의 순서 정보로, -1입력시 마지막에 추가된 파일을 의미합니다.
value  해당 파일에 설정할 CustomValue 정보를 의미합니다.
uploadID  적용할 업로드의 id값을 의미합니다.

remarks

없음.
