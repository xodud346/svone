K Upload :: GetNewUploadList

object GetNewUploadList(type, uploadID)

전송 완료 후 새롭게 업로드 된 파일 리스트 정보를 리턴해 주는 함수 입니다.

return value

Array, Object, xml, text  전송 완료 후 새롭게 업로드 된 파일 리스트를 의미합니다.

parameters

type  리턴받을 형식입니다. ('array', 'json', 'xml', 'text')
uploadID  파일 리스트를 가져올 업로드의 id를 의미합니다.

remarks

없음.
