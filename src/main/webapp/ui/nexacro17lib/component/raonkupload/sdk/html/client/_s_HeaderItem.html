K Upload :: HeaderItem

XML : <header_item><item>.....</item></header_item>, Script : HeaderBarItem
XML : <header_item><item width="....."></item></header_item>, Script : HeaderBarItemWidth

Script 설정이 우선 적용됩니다.
업로드의 상단에 사용자 헤더를 추가 설정합니다.

remarks

XML:item, Script:HeaderBarItem  추가하려는 사용자 헤더의 헤더명을 설정합니다.
XML 설정값은 항목으로 추가, Script 설정값은 콤마(,)로 추가
XML:width, Script:HeaderBarItemWidth  추가하려는 사용자 헤더의 넓이를 설정합니다. (px만 설정 가능)
XML 설정값은 항목으로 추가, Script 설정값은 콤마(,)로 추가

업로드된 파일 추가 시 API 호출 방법  RAONKUPLOAD.AddUploadedFileEx(uniqueKey, originName, webPath, size, customValue, headerEx, uploadName);
기존의 RAONKUPLOAD.AddUploadedFile API와 파라미터는 동일 하고
headerEx에 각 헤더에 맞는 value를 넣어주시면 됩니다.

sample code

업로드의 상단에 사용자헤더1,사용자헤더2 의 이름으로 헤더를 추가합니다.
각각의 넓이는 30px로 설정합니다.
업로드된 파일을 추가할 때 사용자헤더1,사용자헤더2에 테스트1 value,테스트2 value를 설정합니다.

XML

.....
<setting>
    .....
    <header_item>
        <item width="30">사용자헤더1</item>
        <item width="30">사용자헤더2</item>
    </header_item>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->

    function RAONKUPLOAD_CreationComplete(uploadID) {
        ....
        RAONKUPLOAD.AddUploadedFileEx
            ('1', '테스트.txt', 'http://' + location.host + '/sample_data/테스트.txt', '87325', '테스트.txt', '테스트1 value|테스트2 value', uploadID);
        ....
    }
</script>
