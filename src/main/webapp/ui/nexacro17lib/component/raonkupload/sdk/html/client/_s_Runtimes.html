K Upload :: Runtimes

XML : <runtimes>.....</runtimes>, Script : Runtimes

Script 설정이 우선 적용됩니다.
업로드 모드를 설정합니다.
(Agent모드, Html5모드 중 선택)

remarks

agent 설정(기본값)시 모든 브라우저가 Agent모드로 동작됩니다.

html5 설정시 Html5가 지원되는 브라우저는 Html5모드로,
지원되지 않는 브라우저는 Agent모드로 동작됩니다.

sample code

업로드를 Agent 모드로 설정합니다.

XML

.....
<setting>
    .....
    <runtimes>agent</runtimes>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            Runtimes: 'agent'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
