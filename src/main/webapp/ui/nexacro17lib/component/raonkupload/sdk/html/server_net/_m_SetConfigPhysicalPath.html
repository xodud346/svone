K Upload :: SetConfigPhysicalPath

void SetConfigPhysicalPath(String pConfigPhysicalPath)

환경설정 파일을 서버에서 관리하고, 서버에 저장된 환경설정 파일을 사용할 경우 사용합니다.

return value

없음.

parameters

pConfigPhysicalPath  서버에 저장된 환경설정 파일의 경로를 의미합니다.

remarks

기본값은 사용 안 함입니다.

sample code

Client

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // Full URL은 K Upload Handler 경로로 지정하며,
        // f 파라미터의 값은 서버에서 처리할 XML 파일명을 지정합니다.
        var raonkParam = {
            Id: 'raonkupload',
            InitServerXml: 'http://localhost/handler/raonkhandler.ashx?f=raonkupload.config.xml'
        }
        new RAONKUpload(raonkParam);
    </script>
</div>

Server

RAONKHandler upload = new RAONKHandler();
upload.settingVo.SetConfigPhysicalPath(@"D:\config");
upload.Process(context);
