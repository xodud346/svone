K Upload :: BeforeDeleteFile

boolean RAONKUPLOAD_BeforeDeleteFile(uploadID, paramObj)

파일 삭제 전 발생합니다.

return value

boolean  true일 경우 파일을 삭제하고, false일 경우 삭제하지 않습니다.

parameters

uploadID  파일 삭제 요청한 업로드의 id를 의미합니다.
paramObj  삭제될 파일에 대한 정보가 있는 object 입니다.
- paramObj.strIsWebFile  Web파일과 Local 파일의 구분 값을 의미합니다. (0:local, 1:web)
- paramObj.strKey  파일의 Unique Key를 의미합니다.
- paramObj.nIndex  0부터 시작되는 파일의 순서 정보를 의미합니다.
- paramObj.strPath  파일의 경로를 의미합니다.
- paramObj.nSize  파일의 사이즈를 의미합니다.
- paramObj.isFolder  삭제된 항목이 파일인지 폴더인지를 의미합니다. (true:폴더, false:파일)

remarks

K Upload 설정 config.xml에서 use_delete_event의 값이 1일 경우 발생합니다.
