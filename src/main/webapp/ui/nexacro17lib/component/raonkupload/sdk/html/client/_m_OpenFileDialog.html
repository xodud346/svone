K Upload :: OpenFileDialog

void OpenFileDialog(uploadID, isFolderAdd)

파일 선택 다이얼로그 화면 혹은 폴더 선택 다이얼로그 화면을 보여줍니다.

[Agent 전용] 폴더 선택 다이얼로그로 사용 시, isFolderAdd를 true로 설정합니다.

return value

없음.

parameters

uploadID  접근할 업로드의 id를 의미합니다.

isFolderAdd  폴더로 추가시 true로 설정합니다.

remarks

없음.

        }
