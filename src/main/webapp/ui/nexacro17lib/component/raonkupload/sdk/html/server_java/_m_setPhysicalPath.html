K Upload :: SetPhysicalPath

void setPhysicalPath(String pPhysicalPath)

실제 업로드 할 기본경로를 물리적 경로로 설정합니다.
폴더명 제일 뒤에 '.'과 공백이 있다면 제거하시고 설정해 주세요.(운영체제에서 지원되지 않는 문자열입니다.)

return value

없음.

parameters

pPhysicalPath  실제 업로드 할 물리적 경로를 의미합니다.

remarks

기본값은 ""이고, setPhysicalPath 또는 setVirtualPath 중에 반드시 하나는 설정해야 업로드가 가능합니다.
setPhysicalPath의 경로가 공유 드라이브 일 경우 setTempPath도 해당 공유 드라이브의 위치로 설정되어야 합니다.

sample code

RAONKHandler upload = new RAONKHandler();

upload.settingVo.setPhysicalPath("D:\\kupload\\raonkuploaddata");

String result = upload.Process(request, response, application, event);
