K Upload :: SelectItem

void RAONKUPLOAD_SelectItem(uploadID, paramObj)

파일 선택 후 발생합니다.

return value

없음.

parameters

uploadID  파일을 선택한 업로드의 id를 의미합니다.
paramObj  선택한 파일에 대한 정보가 있는 object 입니다.
- paramObj.colunmIndex  파일 선택이 발생한 colunmIndex를 의미합니다.
- paramObj.isWebFile  파일이 웹파일이면 "y" 아니면 "n"을 의미합니다.
- paramObj.nFileSize  파일의 사이즈를 의미합니다.
- paramObj.nIndex  파일 리스트에서의 순서를 의미합니다.
- paramObj.strCustomValue  파일의 CustomValue를 의미합니다.
- paramObj.strExtension  파일의 확장자를 의미합니다.
- paramObj.strLocalPath  파일의 로컬 경로를 의미합니다.
- paramObj.strOriginalName  파일의 원본 파일명을 의미합니다.
- paramObj.strUniqkey  파일의 Uniqkey를 의미합니다.
- paramObj.strWebPath  파일의 웹 경로를 의미합니다.
- paramObj.objFile  파일의 객체 정보를 의미합니다.

remarks

없음.
