K Upload :: ShowViewAlign

XML : <show_button_bar_view align="....."></show_button_bar_view>, Script : ShowViewAlign

Script 설정이 우선 적용됩니다.
업로드가 보기모드일 때 버튼 위치를 설정합니다.

remarks

기본값이 "left" 왼쪽 정렬이고, "right"는 오른쪽 정렬입니다.

sample code

업로드가 보기모드일 때 버튼을 오른쪽 정렬합니다.

XML

.....
<setting>
    .....
    <show_button_bar_view align="right"></show_button_bar_view>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ShowViewAlign: 'right'
        }

    </script>
</div>
