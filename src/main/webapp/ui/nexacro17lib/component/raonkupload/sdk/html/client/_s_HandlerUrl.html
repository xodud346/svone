K Upload :: HandlerUrl

XML : <handler_url>.....</handler_url>, Script : HandlerUrl

Script 설정이 우선 적용됩니다.
웹에서 RFC 1867 표준에 따르는 POST 방식의 파일 전송을 받아주는 페이지 URL입니다.
이미지 파일 또는 Flash, media 파일 등 서버에 파일이 업로드 되지 않을 경우 이 값의 설정을 확인하세요.

remarks

기본값은 "" 입니다.
raonkpload/handler/ 아래의 각 언어에 맞는 페이지를 호출합니다.
상대 및 절대 경로 모두를 지원합니다.
각 개발 언어에 맞는 값으로 설정하세요.

sample code

업로드 handler url을 설정합니다.

XML

.....
<uploader_setting>
    .....
    <handler_url>http://www.raonk.com/handler/raonkhandler.jsp</handler_url>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            HandlerUrl: 'http://www.raonk.com/handler/raonkhandler.jsp'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
