K Upload :: ConvertImageSize

void ConvertImageSize(string pSourceFileFullPath, int pNewWidth, int pNewHeight, long pJpegQuality)

업로드되는 파일이 이미지 파일 일 때, 경우 이미지 크기를 변환할 경우 사용됩니다.
이벤트 AfterUploadEvent 에서 사용이 가능합니다.

return value

없음.

parameters

pSourceFileFullPath  이미지 파일의 물리적 경로를 의미합니다.
pNewWidth  이미지의 크기를 변환할 Width를 의미합니다.
pNewHeight  이미지의 크기를 변환할 Height를 의미합니다.
pJpegQuality  JPG 품질을 의미합니다.(1 ~ 100)

remarks

없음.

sample code

string strNewFileLocation = eventVo.GetNewFileLocation();
try
{
    long lJpegQuality = 100; // JPG 품질 (1 ~ 100)
    string strSourceFileFullPath = strNewFileLocation;
        
    ImageApi.ConvertImageSize(strSourceFileFullPath, 500, 30, lJpegQuality);

}
catch (Exception ex) { }
