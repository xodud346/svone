K Upload :: FileMoveContextMenu

XML : <file_move_context_menu>.....</file_move_context_menu>, Script : FileMoveContextMenu

Script 설정이 우선 적용됩니다.
파일이동 메뉴를 우클릭 메뉴에 추가할지 여부를 설정합니다.

remarks

기본값은 "0" 추가안함이고, "1"이면 파일 이동 메뉴를 우클릭 메뉴에 추가합니다.

sample code

파일이동 메뉴를 우클릭 메뉴에 추가합니다.

XML

.....
<setting>
    .....
    <file_move_context_menu>1</file_move_context_menu>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            FileMoveContextMenu: '1'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
