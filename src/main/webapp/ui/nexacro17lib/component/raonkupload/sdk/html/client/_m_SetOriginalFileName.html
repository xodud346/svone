K Upload :: SetOriginalFileName

void SetOriginalFileName(itemIndex, fileName, uploadID)

리스트에 추가된 파일명을 변경합니다.

return value

없음.

parameters

itemIndex  변경할 파일의 Index를 의미합니다.
fileName  변경할 파일명을 의미합니다.
uploadID  실행하려는 업로드의 id값을 의미합니다.

remarks

없음.
