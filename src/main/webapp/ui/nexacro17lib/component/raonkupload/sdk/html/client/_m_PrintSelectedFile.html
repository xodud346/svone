K Upload :: PrintSelectedFile

void PrintSelectedFile(uploadID)

리스트의 파일 중 선택된 파일을 인쇄합니다.

return value

없음.

parameters

uploadID  실행할 업로드의 id를 의미합니다.

remarks

업로드 모드가 view일 때 동작 합니다. (보기모드)

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function printSelectedFile() {
            RAONKUPLOAD.PrintSelectedFile("upload1");
        }
</script>	

<button type="button" onclick="printSelectedFile()">프린트</button>
 
<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new RAONKUpload({Id: 'upload1'});
        </script>
</div>
