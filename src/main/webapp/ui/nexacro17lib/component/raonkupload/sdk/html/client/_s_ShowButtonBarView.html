K Upload :: ShowButtonBarView

XML : <show_button_bar_view>.....</show_button_bar_view>, Script : ButtonBarView

Script 설정이 우선 적용됩니다.
업로드가 보기모드일 때 버튼바 영역에 노출 할 버튼을 설정합니다.

remarks

"0"으로 설정시 버튼 영역을 표시하지 않습니다.

버튼 종류는
open : 열기
download : 다운로드
download_all : 전체 다운로드
saveandopen : 저장후 열기(Agent 전용)
print : 인쇄(Agent 전용)

버튼 종류는 ",(콤마)"로 구분하여 설정합니다.

sample code

업로드가 보기모드 일 때 버튼 바 영역에 파일열기,다운로드 버튼을 설정합니다.

XML

.....
<setting>
    .....
    <show_button_bar_view>open,download</show_button_bar_view>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ButtonBarView: 'open,download'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
