K Upload :: AllowedZeroFileSize

XML : <allowed_zero_file_size>.....</allowed_zero_file_size>, Script : AllowedZeroFileSize

Script 설정이 우선 적용됩니다.
0byte 파일의 업로드를 허용 할지 설정합니다.

remarks

기본값은 "0" 허용안함이고, "1"로 설정시 허용입니다.

sample code

0byte 파일의 업로드를 허용으로 설정합니다.

XML

.....
<setting>
    .....
    <allowed_zero_file_size>1</allowed_zero_file_size>
    .....
</setting>
.....
