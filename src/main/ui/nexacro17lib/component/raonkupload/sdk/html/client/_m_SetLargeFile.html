K Upload :: SetLargeFile

void SetLargeFile(fileIdx, boolLarge, uploadID)

파일의 대용량 설정을 적용 또는 해제합니다.

return value

없음.

parameters

fileIdx   0부터 시작되는 파일의 순서정보를 의미합니다.
boolLarge   true 입력 시 대용량 설정 적용, false 입력 시 해제를 의미합니다.
uploadID  대용량 설정을 적용할 업로드의 id값을 의미합니다.

remarks

없음.

        function setLargeFile() {
            RAONKUPLOAD.SetLargeFile('0', true, 'upload1');
        }
</script>
