K Upload :: AllowDuplicationFile

XML : <allow_duplication_file>.....</allow_duplication_file>, Script : AllowDuplicationFile

Script 설정이 우선 적용됩니다.
파일 추가 시 중복 파일 허용 여부를 설정합니다.

remarks

기본값은 '0'으로 중복 허용을 안합니다.

sample code

    .....

    .....
