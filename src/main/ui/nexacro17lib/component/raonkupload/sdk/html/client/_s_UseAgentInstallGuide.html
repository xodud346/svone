[Agent 전용]

K Upload :: UseAgentInstallGuide

XML : <use_agent_install_guide>.....</use_agent_install_guide>, Script : UseAgentInstallGuide
XML : <use_agent_install_guide zindex="....."></use_agent_install_guide>, Script : AgentInstallGuideZIndex

Script 설정이 우선 적용됩니다.
Agent 설치시 설치 안내창 노출여부를 설정합니다.

remarks

XML:use_agent_install_guide, Script:UseAgentInstallGuide  기본값이 "1" Agent 설치시 설치 안내창 노출이고, "0"은 노출하지 않습니다.
XML:zindex, Script:AgentInstallGuideZIndex  Agent 설치 안내창에 대한 ZIndex를 설정합니다. (기본값: 1000)

sample code

Agent 설치시 설치 안내창 노출여부를 노출로 설정합니다.
Agent 설치 안내창의 ZIndex는 1500으로 설정합니다.

XML

.....
<setting>
    .....
    <use_agent_install_guide zindex="1500">1</use_agent_install_guide>
    .....
</setting>
.....
