K Upload :: setViewerBrokenImagePath

void setViewerBrokenImagePath(String pViewerBrokenImagePath)

웹 표준 모드의 뷰어 사용시 뷰어에 사용되는 kupload/images/broken.png 파일의 경로를 설정합니다.
일반적으로 kupload/handler/raonkviewer.jsp를 다른 경로에 사용될 경우 설정합니다.
이 경우 raonkviewer.jsp 페이지에서 broken.png의 경로가 접근 가능하도록 설정해야 합니다.

return value

없음.

parameters

pViewerBrokenImagePath  웹 표준 모드의 뷰어 사용시 뷰어에 사용되는 kupload/images/broken.png 파일의 경로를 의미합니다.

remarks

기본값 : ../images/broken.png ( kupload/images/broken.png )

sample code

//raonkviewer.jsp에서 설정

RAONKHandler upload = new RAONKHandler();

upload.settingVo.setViewerBrokenImagePath("../images/broken.png");

String result = upload.Process(request, response, application, event);
