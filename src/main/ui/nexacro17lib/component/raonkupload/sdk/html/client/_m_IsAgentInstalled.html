K Upload :: IsAgentInstalled

boolean IsAgentInstalled(fn_callback)

설치가 되었는지 확인합니다.

return value

boolean  설치가 되었으면 true, 설치가 되지 않았다면 false 값을 리턴합니다.

parameters

fn_callback  콜백 함수를 의미합니다.

remarks

RAONKUPLOAD.IsAgentInstalled(fn_callback); 호출 후 콜백 함수 안에서 설치 여부에 따른 필요 로직을 구현하시면 됩니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        var fn_callback = function (value) {
            // value가 true이면 설치됨
            // value가 false이면 설치되지 않음
        };

        function isAgentInstalled() {
            RAONKUPLOAD.IsAgentInstalled(fn_callback);
        }
</script>
