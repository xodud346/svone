K Upload :: Views

XML : <mode views="....."></mode>, Script : Views

Script 설정이 우선 적용됩니다.
업로드 파일 목록 보기 형태를 설정합니다.

remarks

기본값은 "list" 입니다.
"list"로 설정시 "리스트형태로 보기", "thumbs" 로 설정시 "썸네일형태로 보기" 입니다.

sample code

업로드 파일 목록 보기 형태를 썸네일보기로 설정합니다.

XML

.....
<setting>
    .....
    <mode views="thumbs"></mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            Views: 'thumbs'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
