K Upload :: SetDefaultDownloadPath

void SetDefaultDownloadPath(path, uploadName)

다운로드 경로를 설정합니다.

return value

없음.

parameters

path  다운로드 경로(물리적 경로), 빈 string 값인 경우 다운로드 경로가 삭제 됩니다.
uploadID  설정할 업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="upload1/js/raonkupload.js"></script>	
<script type="text/javascript">
        function setDefaultDownloadPath() { 
            // c:\temp\folder1 이라는 폴더에 파일이 저장
            RAONKUPLOAD.SetDefaultDownloadPath('c:\\temp\\folder1', 'upload1');

            // 빈 string 값으로 설정하는 경우 경로 설정 값이 삭제되고, 파일 다운로드 시 경로를 선택하기 위한 dialog 창이 나타납니다.
            RAONKUPLOAD.SetDefaultDownloadPath('', 'upload1');
        }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new RAONKUpload({Id: 'upload1'});
        </script>
</div>
