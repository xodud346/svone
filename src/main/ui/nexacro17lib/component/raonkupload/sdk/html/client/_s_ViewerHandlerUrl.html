K Upload :: ViewerHandlerUrl

XML : <viewer_handler_url>.....</viewer_handler_url>, Script : ViewerHandlerUrl

Script 설정이 우선 적용됩니다.
웹표준 모드에서 열기 또는 파일리스트를 더블클릭하여 열리는 Viewer페이지를 구현하여 사용하고자 할 때 url을 설정합니다.

remarks

설정하실 때 http:// 부터의 전체경로를 설정합니다.

sample code

업로드 Viewer handler url을 설정합니다.

XML

.....
<uploader_setting>
    .....
    <viewer_handler_url>http://www.raonk.com/handler/raonkviewer.jsp</viewer_handler_url>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ViewerHandlerUrl: 'http://www.raonk.com/handler/raonkviewer.jsp'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
