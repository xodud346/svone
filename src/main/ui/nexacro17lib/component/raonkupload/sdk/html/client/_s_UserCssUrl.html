K Upload :: UserCssUrl

XML : <user_css_url>.....</user_css_url>, Script : UserCssUrl

Script 설정이 우선 적용됩니다.
사용자가 만든 css를 업로드 영역에 설정합니다.

remarks

http://를 포함하는 전체 url을 입력해야 합니다.

sample code

"http://www.raonk.com/user_css.css" 링크의 css를 업로드 영역에 설정합니다.

XML

.....
<setting>
    .....
    <user_css_url>http://www.raonk.com/user_css.css</user_css_url>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UserCssUrl: 'http://www.raonk.com/user_css.css'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
