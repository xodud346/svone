K Upload :: CompleteEventResetUse

XML : <complete_event_reset_use>.....</complete_event_reset_use>, Script : CompleteEventResetUse

Script 설정이 우선 적용됩니다.
업로드 완료 이벤트에서 ResetUpload API 사용여부를 설정합니다.

remarks

설정값 "0"은 사용안함이고, "1"로 설정시 사용입니다.

sample code

.....
<setting>
    .....
    <complete_event_reset_use>1</complete_event_reset_use>
    .....
</setting>
.....
