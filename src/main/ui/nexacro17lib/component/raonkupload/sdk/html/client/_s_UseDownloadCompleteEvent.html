K Upload :: UseDownloadCompleteEvent

XML : <use_download_complete_event>.....</use_download_complete_event>
Script : UseDownloadCompleteEvent

Script 설정이 우선 적용됩니다.
업로드 리스트 컨트롤에서 파일 다운로드 후 이벤트 함수 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 이벤트 미사용합니다.
"1"으로 설정 시 파일 다운로드 후 RAONKUPLOAD_DownloadCompleteFile(uploadID, paramObj) 함수가 호출 됩니다.

sample code

파일 다운로드 후 이벤트 함수 미사용으로 설정합니다.

    .....
    <use_download_complete_event>0</use_download_complete_event>
    .....
</setting>
