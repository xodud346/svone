K Upload :: GetSelectedFileCount

int GetSelectedFileCount(uploadID)

업로드 영역에서 선택된 항목수를 얻어옵니다.

return value

int  업로드 영역의 선택된 항목수를 의미합니다. 업로드 영역에 파일이 없을 경우 -1을 리턴합니다.

parameters

uploadID  항목수를 얻어올 업로드의 id를 의미합니다.

remarks

없음.

       var file_cnt = RAONKUPLOAD.GetSelectedFileCount('upload1');
</script>
