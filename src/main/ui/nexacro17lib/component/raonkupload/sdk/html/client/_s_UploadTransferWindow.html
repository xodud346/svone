K Upload :: UploadTransferWindow

XML : <upload_transfer_window><view>.....</view></upload_transfer_window>
Script : UploadTransferWindow.View

Script 설정이 우선 적용됩니다.
업로드 전송창 UI를 설정합니다.

remarks

XML:view, Script:View  기본값은 "standard" 이고, 축소창은 "light" 입니다.
Standard 창은 전체용량 + 현재파일 용량을 보여주고,
Light 창은 전체용량만,
Advanced 창은 전체용량 + 모든전송파일 용량을 보여줍니다.

sample code

업로드 전송창을 Light 창으로 설정합니다.

XML

.....
<setting>
    .....
    <upload_transfer_window>
        <view>0</view>
        .....
    </upload_transfer_window>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....

            }
        }

        new RAONKUpload(raonkParam);

</div>
