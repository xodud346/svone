서버에서 환경설정 파일 읽기

K Upload의 기능에 대한 설정은 기본적으로 XML에 설정하게 되어있습니다.
만일 XML에 파일 저장 경로등과 같은 값의 노출이 문제가 되는 경우 XML파일을 암호화 하여 처리 하는 방식을 의미합니다.

remarks

없음.

방법 설명

1. Upload handler 파일안에 환경설정파일(xml)의 물리적 폴더를 지정해주셔야 합니다. (서버 환경설정 파일 읽기를 사용할 경우)

SetConfigPhysicalPath = "환경설정파일(xml)이 존재하는 서버의 폴더 경로"

.NET : upload.settingVo.SetConfigPhysicalPath("D:\\config");
JAVA : upload.settingVo.setConfigPhysicalPath("D:\\config");


2. 폴더 지정 후 업로드 해당 페이지에 환경 설정 값을 적용 합니다.

<script type="text/javascript">

var raonkParam = {
    .....
    InitServerXml: 'http://localhost/handler/raonkhandler.jsp?f=raonkupload.config.xml'
}

new RAONKUpload(raonkParam);
</script>

※ InitServerXml의 환경설정은 Full URL(http 포함)로 다음과 같이 설정합니다.
Full URL은 K Upload Handler경로로 지정하며, f 파라미터의 값은 서버에서 처리할 XML 파일명을 지정합니다.
예시)
var raonkParam = {
    .....
    InitServerXml: 'http://localhost/raonkupload/handler/raonkhandler.jsp?f=raonkupload.config.xml'
}
