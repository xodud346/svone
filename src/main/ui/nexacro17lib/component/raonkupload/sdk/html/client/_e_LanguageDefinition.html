K Upload :: LanguageDefinition

void RAONKUPLOAD_OnLanguageDefinition(uploadID, uploadLang)

업로드 생성 시 이벤트가 발생합니다.

return value

없음.

parameters

uploadID  업로드의 id입니다.
uploadLang  다국어 설정값(lang)에 따른 텍스트 객체입니다.

remarks

업로드에서 노출되는 버튼명, Text 등을 변경합니다.
한글, 영어, 일본어, 중국어(간체,번체)에 대한 별도의 다국어 설정이 존재합니다.
