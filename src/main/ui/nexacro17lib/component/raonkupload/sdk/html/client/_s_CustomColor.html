K Upload :: CustomHeaderColor, CustomFooterColor
K Upload :: CustomProgressBarColor, CustomTextColor

XML : <skin_name color1="....."></skin_name>, Script : CustomHeaderColor
XML : <skin_name color2="....."></skin_name>, Script : CustomFooterColor
XML : <skin_name color3="....."></skin_name>, Script : CustomProgressBarColor
XML : <skin_name text_color="....."></skin_name>, Script : CustomTextColor

Script 설정이 우선 적용됩니다.
업로드 색상을 사용자가 원하는 색상으로 변경합니다.

remarks

CustomHeaderColor  업로드창 (상단/ 하단 / 하단 버튼 바탕색)을 설정합니다.
CustomFooterColor   업로드창 (상단 / 하단 버튼 텍스트색)을 설정합니다.
CustomProgressBarColor  전송창(상단/ 전체 전송률/ 전송 용량률/ 취소 버튼 바탕색)을 설정합니다.
CustomTextColor  전송창 (상단 / 하단 취소 버튼 텍스트 색)을 설정합니다.

#을 포함한 hex값으로 입력합니다.

sample code

사용자가 원하는 색상으로 설정합니다.

XML

.....
<setting>
    .....
    <skin_name color1="#ff0000" color2="#f7f140" color3="#33e8f5" text_color="#c2ffd0">.....</skin_name>
    .....
</setting>
.....

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
