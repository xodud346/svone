K Upload :: ImgPreView

XML : <img_preview>.....</img_preview>, Script : ImgPreView
XML : <img_preview width="....."></img_preview>, Script : ImgPreViewWidth
XML : <img_preview height"....."></img_preview>, Script : ImgPreViewHeight

Script 설정이 우선 적용됩니다.
이미지 미리보기 창 기능을 설정합니다.

remarks

기본값은 "0"이고 사용안함, "1"로 설정시 이미지 미리보기 창 사용입니다.

XML:width, Script:ImgPreViewWidth  이미지 미리보기 창의 넓이를 설정합니다.
기본값은 "320px" 입니다.
넓이는 "px"(픽셀)만 가능합니다.
XML:height, Script:ImgPreViewHeight  이미지 미리보기 창의 높이를 설정합니다.
기본값은 "280px" 입니다.
높이는 "px"(픽셀)만 가능합니다.

sample code

이미지 미리보기 창 사용으로 설정합니다.
이미지 미리보기 창 넓이를 250px 로 설정합니다.
이미지 미리보기 창 넓이를 200px 로 설정합니다.

XML

.....
<setting>
    .....
    <img_preview width="250px" height="200px">1</img_preview>
    .....
</setting>
.....
