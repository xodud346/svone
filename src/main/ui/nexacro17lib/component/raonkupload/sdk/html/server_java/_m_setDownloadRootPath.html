K Upload :: setDownloadRootPath

void setDownloadRootPath(String pDownloadRootPath)

클라이언트에서 다운로드 또는 열기 경로 설정시 경로의 앞부분 일부를 서버에서 설정할 경우 사용합니다.
다운로드 또는 열기의 모든 경로에 고정적으로 설정되므로 모든 파일의 경로가 동일할 경우 사용해야 합니다.

return value

없음.

parameters

pDownloadRootPath  클라이언트에서 다운로드 또는 열기 경로 설정시 경로의 앞부분 일부의 특정 경로를 의미합니다.

remarks

기본값 : 빈값

sample code

Client

<script type="text/javascript">
    // 실제 경로는 /download/전경사진.jpg 이라고 가정
    RAONKUPLOAD.AddUploadedFile('1', '전경사진.jpg', '/전경사진.jpg', '87325', '전경사진.jpg', 'raonkupload');
</script>

Server

RAONKHandler upload = new RAONKHandler();
upload.settingVo.setDownloadRootPath("/download");
String result = upload.Process(request, response, application, event);
