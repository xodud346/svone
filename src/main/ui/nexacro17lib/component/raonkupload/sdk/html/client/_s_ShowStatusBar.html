remarks

기본값은 "1" 이고, "0" 으로 설정시 상태바를 표시하지 않습니다.

XML:show_limit, Script:StatusBarShowLimit  "파일 제한" 문구를 표시하거나 숨기는 기능을 설정합니다.
기본값은 "1" 이고, "0" 으로 설정시 "파일 제한" 문구를 표시하지 않습니다.
XML:show_status, Script:StatusBarShowStatus  "현재 파일 상태" 문구를 표시하거나 숨기는 기능을 설정합니다.
기본값은 "1" 이고, "0" 으로 설정시 현재 파일 상태 문구를 표시하지 않습니다.
