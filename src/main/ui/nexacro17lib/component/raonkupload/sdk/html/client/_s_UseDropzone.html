K Upload :: UseDropzone

XML : <use_dropzone>.....</use_dropzone>, Script : UseDropzone

Script 설정이 우선 적용됩니다.
Drop Zone 이벤트 함수 사용을 설정합니다.

remarks

DropZone설정값은 다음과 같습니다.
XML - use_dropzone
Script - UseDropzone

기본값은 "0" 미사용이고, "1"로 설정시 Drop Zone으로 만들려는 객체의 클래스명이 "raonkdropzone"을 포함하고 있다면 Drop Zone으로 만듭니다.

sample code

Drop Zone 이벤트 함수 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <use_dropzone>1</use_dropzone>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->

<div id="upload1_dropzone" class="raonkdropzone" style="width: 800px; height: 60px; padding-top: 40px">Drop Zone</div>

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseDropzone: '1'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
